Project 1: Movie Trailer Website

To run the project, you will need the following files:
- movie_class.py:
    collection of classes required to store all movie information

- entertainment_center.py:
    1. instantiate Movie objects for all the movies that will be displayed
    2. call the function to generate the HTML code
    3. open the HTML code in a browser

- fresh_tomatoes.py:
    1. contains the HTML/CSS code to create the webpage
    2. contains the code - create_movie_tiles_content() to combine the HTML/CSS along with the movie info
    3. contains the code - open_movies_page() - to write the final HTML/CSS to the .html file and open it in a browser

Additional requirements are:
- Python 2.7.3 or above
- modern web browser
